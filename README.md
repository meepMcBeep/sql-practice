# SQL Practice

This repository contains structured SQL exercises designed to rebuild and strengthen core database and query skills.

The focus is on:
- Clean SQL syntax
- Progressive complexity
- Real-world query patterns
- Consistent daily practice

---

## 📚 Structure

Exercises are organized into **sets**, each focusing on a specific concept:

sql/
├── set1/ -- Basic queries (SELECT, WHERE, ORDER BY)
├── set2/ -- Aggregation (COUNT, AVG, GROUP BY, HAVING)
├── set3/ -- Joins (multi-table queries)
├── set4/ -- (upcoming)


Each file represents a single assignment and includes:
- The problem statement (as a comment)
- The SQL solution

---

## 🎯 Goals

- Rebuild SQL fluency from fundamentals
- Develop clean and readable query habits
- Understand relational data modeling
- Practice real-world query patterns used in analytics and backend systems

---

## 🧠 Learning Approach

- Small, focused exercises (10–15 minutes)
- Emphasis on **recall over recognition**
- Minimal hints to encourage independent problem solving
- Iterative improvement through review and refinement

---

## 🛠️ Environment

Exercises are tested using:
- SQLite (browser-based tools like DB Fiddle / SQLiteOnline)
- Future expansion to PostgreSQL for more advanced features

---

## ✍️ SQL Style Guidelines

- Use explicit column selection (avoid `SELECT *`)
- Format queries vertically for readability
- Capitalize SQL keywords (`SELECT`, `FROM`, etc.)
- Use meaningful aliases when appropriate
- End statements with semicolons
